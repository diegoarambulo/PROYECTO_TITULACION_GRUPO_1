# 📄 manual de usuario – sisad web

## 🧾 1. introduccion

### 🎯 1.1 objetivo del manual

este manual proporciona la guia necesaria para el uso correcto de la plataforma **sisad web**, permitiendo al usuario gestionar documentos, realizar consultas y administrar informacion de manera eficiente.

### 👥 1.2 audiencia

dirigido a usuarios del cliente que utilizan la plataforma para:

* consulta de documentos
* administracion de registros
* visualizacion de informacion

---

## 🌐 2. descripcion general del sistema

sisad web es una aplicacion disenada para:

* almacenar documentos digitales
* organizar informacion mediante indices (metadatos)
* gestionar documentos via web
* clasificar informacion por fechas, carpetas y estructuras jerarquicas

## 🖥 7. vista tradicional

es la principal vista operativa del sistema donde se gestionan registros y documentos.

---

## 🧩 7.1 elementos de la vista tradicional

### ➕ boton crear

permite crear un nuevo registro ingresando metadata.

---

### ✏ boton editar

permite modificar la metadata de un registro.

---

### ❌ boton eliminar

permite eliminar registros.

⚠ **condicion:**
solo si el registro no tiene archivos asociados.

---

### 📊 reporte con filtros avanzados

permite generar archivos excel con:

* filtros por fechas
* cantidad de registros
* inclusion de urls

✔ puede tardar segun volumen de datos

---

### 🔄 actualizacion masiva

* permite modificar multiples registros
* se realiza mediante archivo excel

---

### ⬆ subida masiva

#### 📄 metadata

pasos:

1. descargar plantilla
2. completar datos
3. subir archivo

⚠ formato de fecha: `-mm-dd`

---

#### 📎 archivos

* subir multiples archivos
* el nombre debe coincidir con el id sisad

campos en plantilla:

* archivo
* tipo de archivo
* nombre (opcional)
* descripcion (opcional)

---

### ⬇ descarga masiva

opciones:

1. descargar registros seleccionados
2. descargar todos los registros
3. descargar sin filtros

✔ se puede configurar nombre del archivo con metadatos

---

### 📧 compartir archivos

permite enviar documentos por correo.

configuraciones:

* vigencia del enlace
* tipos de archivo
* correos en copia (cc)

⚠ requiere configuracion previa

---

### 📊 tipos de archivo como columnas

permite visualizar:

* cantidad de archivos por tipo
* distribucion por registro

✔ mejora trazabilidad

---
